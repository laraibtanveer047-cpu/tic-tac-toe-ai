# Tic Tac Toe AI Game 🎮

## Introduction

This is a simple Tic Tac Toe game where a human plays against an AI.

## AI Logic

The AI uses the Minimax algorithm to make optimal decisions.

## Features

* Player vs AI
* Smart AI (never loses)
* Console-based game

## How to Run

1. Download the code
2. Run in Python:
   python game.py
